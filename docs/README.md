# Documentación — UNparche

UNparche es una aplicación móvil diseñada exclusivamente para la comunidad universitaria de la Universidad Nacional de Colombia (sede Bogotá). Centraliza la difusión y el descubrimiento de eventos académicos, culturales, deportivos y sociales del campus, integrando un mapa interactivo con geolocalización y un chat en tiempo real por evento.
